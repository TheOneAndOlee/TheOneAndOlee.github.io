A Lot of Learning!

It's been 3+ weeks since my last update, and since I'm feeling really good after my CS test, I decided to post an update!

Honestly not much has happened IRL, but I did spend a lot of time with friends. As for the calculus test, I got a 84, and honestly, I'm pretty proud of myself. I've made progress from my previous calculus tests since I actually understand my mistakes (even if it doesn't make me happy.)

As for stuff I've managed to do in Unity, I've added the following:

  1. A healthbar - Done! Instead of a healthbar, I chose to display health as a text (because it provides more visual clarity).
  2. Getting EXP from enemies - Done! It took me a bit of debugging, but killing enemies does grant the player exp.
  3. Leveling up - Done! I've added a rudimentary EXP curve that will definitely change, but exp logic itself has been added.
I also changed some other things, and currently, I'm working on adding an inventory system following this guide from Night Run Studio. ' Some other changes I made were:
  1. Changing the font from the default to Lexend in the game.
  2. Added actual spawning logic that spawns more and more enemies as time passes.

Since I've pretty much added the core mechanics of any survival game, I'll be moving forward with unique game mechanics. I've got some work to do, especially in real life (another calculus test hooray!), but my future plans are:

  1. Finish the inventory system
  2. From there, adding the crafting/forging system
  3. Add logic for enemies to drop items
  4. Get blueprints from leveling up (see below).
In the future, I still want to work on:
  1. The 3rd weapon (magic)
  2. Creating/Spawning stronger enemies
To explain the blueprints system: I want the plyaers to be able to get blueprints or materials upon level up. The blueprints allow players to unlock unique variants of their weapon, each bringing their own strengths or downsides. For example, a halberd would give you a bigger swing and more damage, but a slower swing and attack speed. To implement this, I was also considering making it so that your weapon starts in a weaker form, so the sword would start as a stick, and the bow, a slingshot.

This post is gonna be way shorter than the previous one, mainly because it's 1:26 AM, and I desperately need to sleep for my CS tutorial tomorrow at 9:30. Still, I hope I showed you my progress, and thank you!
